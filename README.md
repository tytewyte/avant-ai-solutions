## About

Avant AI Solutions builds intelligent AI chat agents specifically designed for law firms. Our agents work 24/7 to capture leads, qualify cases, and schedule consultations - even when your office is closed.

## Features

- **24/7 Availability**: Never miss a potential client again
- **Intelligent Lead Qualification**: Asks the right questions to determine case type and urgency
- **Automated Contact Collection**: Captures client information and schedules consultations
- **Multi-Practice Area Support**: Criminal Defense, Personal Injury, Family Law, and more
- **Professional & Conversational**: Natural language interactions that build trust

## Technology Stack

- React + TypeScript
- Vite (build tool)
- Modern CSS with responsive design
- Deployable to Netlify/Vercel

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:3000` to see the demo.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
